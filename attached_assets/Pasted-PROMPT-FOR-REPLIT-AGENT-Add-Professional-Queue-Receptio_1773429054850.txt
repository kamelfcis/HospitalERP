PROMPT FOR REPLIT AGENT
Add Professional Queue / Reception Ticket Printing (Patient Reception Only)

You are implementing a small professional reception ticket printing feature in the Hospital ERP system.

This feature is for Patient Reception inside the Patient Record / Patient Registry workflow, NOT for bed-board admission.

The ticket is intended for quick printing on a small thermal printer (80mm or 58mm).

The purpose is to give the patient a queue / reception slip that shows key visit information in a clean minimal layout.

CRITICAL RULES

This task is UI + printing only.

FEATURE

Add a toggle option during reception:

Arabic label example:

When enabled, the system prints a small ticket immediately after reception is saved.
