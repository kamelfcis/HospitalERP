🔥 5 ملاحظات نهائية (Production Hardening)
1️⃣ Bulk Fetch — أتأكد إنها فعلاً مطبقة

أنت ذكرت fix N+1 👍

لكن تأكد فعليًا:

GET /api/items?ids=...
GET /api/services?ids=...

❗ مش:

for (...) fetch(...)

👉 لو لسه مش bulk → لازم يتعدل فورًا

2️⃣ Template Apply = Atomic UI Action

دلوقتي:

كل line async
ممكن يحصل partial apply
الأفضل:
startBatch()
apply all
endBatch()

أو:

disable UI أثناء التطبيق
loading state واضح

👉 علشان:

مايحصلش duplicate clicks
مايحصلش inconsistent state
3️⃣ Stay Engine — لازم Validation قوي

أنت عملت:

quantity = NULL

ممتاز 👌

لكن لازم تضيف:

عند save:
if (line.sourceType === "STAY_ENGINE" && !quantity) {
  throw "عدد الأيام مطلوب"
}

👉 مهم جدًا محاسبيًا

4️⃣ Template Trace — ممتاز بس محتاج تكملة

أنت ضفت:

templateId
templateNameSnapshot

كويس جدًا 👌

أضف كمان:
appliedAt
appliedBy

👉 ده هيفرق جدًا في:

audit

التتبع
