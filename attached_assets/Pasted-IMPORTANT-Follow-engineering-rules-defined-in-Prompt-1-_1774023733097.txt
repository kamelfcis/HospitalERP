ISSUE 2 — DATABASE POOL NOT CONFIGURED

Current State:

* pg.Pool uses default settings
* no timeout
* limited connections
* runaway queries possible

Problem:

* slow queries block system
* pool exhaustion
* system freeze

---

REQUIRED FIX

Configure pool in:

server/db.ts

---

POOL CONFIG

* max: 20
* idleTimeoutMillis: 30000
* connectionTimeoutMillis: 5000

---

QUERY TIMEOUT

Set:

statement_timeout = 30s

Using:

pool.on('connect', ...)

STARTUP DB TEST

* log structured error
* exit process with code 1
