🧠 PERMISSION MODEL REQUIREMENT

The UI must clearly distinguish between:

A. Screen Visibility

Whether the user can see/open the screen at all

B. Screen Actions

What the user can do inside that screen

A user may:

view data but not delete

create but not approve
