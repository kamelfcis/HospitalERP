====================================================================================================
3) Database Schema (Required)
====================================================================================================
A) Warehouses
- WarehouseID PK
- WarehouseCode UNIQUE
- WarehouseNameAR
- IsActive BIT DEFAULT 1

B) Items (minimum fields needed for pilot)
- ItemID PK
- ItemCode UNIQUE
- ItemNameAR
- HasExpiry BIT NOT NULL DEFAULT 0
- MajorUnitName NVARCHAR(20)
- MinorUnitName NVARCHAR(20)
- ConversionFactor DECIMAL(18,4) NOT NULL DEFAULT 1

C) InventoryLots (Must include WarehouseID)
- LotID PK
- ItemID FK
- WarehouseID FK
- ExpiryDate DATE NULL
- ReceivedDate DATE NOT NULL
- PurchasePrice DECIMAL(18,4) NOT NULL
- QtyInMinor DECIMAL(18,4) NOT NULL CHECK(QtyInMinor>=0)
- IsActive BIT DEFAULT 1
Constraints:
- CHECK (ExpiryDate IS NULL OR ExpiryDate >= '2000-01-01')
Indexes:
- IX_Lots_Item_Warehouse_Expiry (ItemID, WarehouseID, ExpiryDate)

D) InventoryLotMovements
- MovementID PK
- LotID FK
- WarehouseID FK
- TxDate DATETIME
- TxType NVARCHAR(10)  -- IN/OUT/TRANSFER
- QtyChangeInMinor DECIMAL(18,4)
- UnitCost DECIMAL(18,4)
- ReferenceType NVARCHAR(20) = 'TRANSFER'
- ReferenceID NVARCHAR(50) = TransferNo
Index:
- IX_Mov_Lot_TxDate (LotID, TxDate)

E) Transfers
TransferHeaders:
- TransferID PK
- TransferNo UNIQUE
- TransferDate
- SourceWarehouseID
- DestinationWarehouseID
- Status (Draft/Posted)
- CreatedAt

TransferLines:
- LineID PK
- TransferID FK
- ItemID
- UnitName (Major/Minor)
- QtyEntered
- QtyInMinor
- CreatedAt

TransferLineAllocations:
- AllocationID PK
- LineID FK
- SourceLotID
- ExpiryDate
- QtyOutInMinor
- PurchasePrice
- DestinationLotID
